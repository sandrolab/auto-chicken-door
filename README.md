# Auto Chicken Door

Automatic door (crepuscular) with lock for chicken coop based on Arduino.

## Main components
- Arduino nano
- Servo 1 - 25kg
- Servo 2 - 9g
- DC-DC stepdown
- Photoresistor
- Buzzer
- Led red
- Led green
- Lever switch SPDT
- Capacitor: 100uF 25V
- Resistors: 10kΩ, 1kΩ, 330Ω (x2)
- Power supply DC12V
